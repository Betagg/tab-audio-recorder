Definition

What is a tab audio recorder?

A tab audio recorder captures sound from a specific browser tab. It is different from a voice recorder, a system audio recorder, and a screen recorder.

Dolphin is made for the moment when you hear useful audio in Chrome and want a clean local file without setting up OBS, QuickTime, Audacity, or a virtual audio route.

Instead of recording your whole screen or desktop sound, Dolphin focuses on the active tab you choose and downloads the result as an MP3 file.

Workflow

How to record audio from a Chrome tab

  1. Open the tab with audio

    Open the Chrome tab that contains the lecture, webinar, podcast, web player, or website audio you want to capture.

  2. Make sure the tab is audible

    Play the audio and check that the tab is not muted. Dolphin records the active tab audio it can hear.

  3. Start Dolphin

    Click Dolphin from the Chrome toolbar and start recording the current tab.

  4. Trim the useful segment

    When the recording is done, use the waveform view to keep the part that matters.

  5. Download the MP3

    Trim the useful segment and save the recording as an MP3 file.

Troubleshooting

If your tab audio recording is silent

Check the active tab

Start recording from the tab that is actually playing audio.

Unmute the tab

A muted tab or silent website player can create a silent recording.

Avoid switching sources

If a site reloads or changes players, stop and start a fresh recording from the current tab.

Protected content

Some DRM-based or protected streams may not be available to browser tab capture.

FAQ

Common questions

How do I record audio from a Chrome tab?

Open the Chrome tab with audio, make sure it is not muted, start Dolphin from the toolbar, stop when finished, trim the useful segment, and download the MP3 file.

Can Dolphin record only one tab?

Yes. Dolphin is designed to capture the active Chrome tab after you start recording.

Can I save the recording as MP3 or WAV?

Dolphin currently saves recordings as MP3 files. WAV export is not included in the current release.

Does the audio leave my browser?

No. Recordings are processed locally in Chrome and downloaded to your device.

Is Dolphin a screen recorder?

No. Dolphin does not record your screen or create video files. It focuses on audio from the active Chrome tab.

What if the tab is muted?

If the tab is muted or not playing audible audio, the recording may be silent. Unmute the tab before recording.

Can it record long sessions?

Dolphin is designed for practical browser audio sessions, but very long recordings depend on Chrome, memory, and device performance.

Does it also record microphone or system audio?

No. Dolphin records the active tab audio only. It is not a microphone or full system audio recorder.

Can I record audio from a website with Dolphin?

Yes, when the website audio is playing in the active Chrome tab and is not protected in a way that blocks browser tab capture.
