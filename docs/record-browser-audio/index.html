Step-by-step

How to record audio from one Chrome tab

  1. Open the browser audio source

    Play the website, course, webinar, podcast, or web player in a Chrome tab.

  2. Start Dolphin on that tab

    Click the extension while the audio tab is active so Dolphin captures the right source.

  3. Listen while recording

    Keep the tab audible and avoid muting or reloading the source while the recording is running.

  4. Stop and trim

    Stop when the useful part is done and trim silence or extra audio with the waveform editor.

  5. Download the MP3

    Save the final recording locally through Chrome downloads.

Choose the right mode

Tab audio, screen recording, microphone, or system audio?

Chrome tab audio

Choose Dolphin when the sound is playing in one Chrome tab and you want an audio-only MP3.

Screen plus audio

Use a screen recorder when the video, cursor, or on-screen steps must be part of the final file. Dolphin does not record video.

Microphone voice

Use a voice recorder when you need narration, room sound, or a conversation. Dolphin does not capture microphone input.

Full system audio

Use a desktop audio tool when sound from several apps must be mixed together. Dolphin records one active browser tab.

Before recording

A quick checklist that prevents silent tab recordings

Confirm the source tab

Click the exact tab that is playing the course, webinar, podcast, interview, or web player before starting Dolphin.

Play a few seconds first

Make sure the player has started and the tab is producing audible sound before you begin recording.

Check mute controls

Verify both Chrome's tab mute state and the website player's own volume control.

Avoid changing players

Do not reload the page or switch the source player during capture. Start a fresh recording if the source changes.

Compared with desktop recording

Record the browser tab, not the whole computer

Traditional audio recording tools often need a virtual audio cable, system audio routing, or a separate editor. That can be overkill when the sound is already playing in one Chrome tab.

Dolphin keeps the capture focused on the browser tab and saves the result as a local MP3.

Recording quality

Keep the source stable, then trim the useful part

Browser tab capture uses the audio produced by the active tab, so it avoids room echo from recording speakers with a microphone. The source still matters: low-volume, interrupted, or protected playback can affect the result.

Let the source play normally while Dolphin records. When you stop, use the waveform editor to remove extra silence at the beginning or end before saving the MP3.

Visible waveform

Use the waveform shape to locate the start and end of the segment you need.

Preview the cut

Listen from the adjusted start point before downloading the final file.

Local MP3

Encode and save the result on your device without uploading the recording.

Limits

What browser audio recording cannot guarantee

Dolphin records audio you have the right to capture from supported Chrome tabs. Some protected or DRM-based streams may block browser tab capture.

If the site reloads, changes players, or mutes the tab, stop and start a fresh recording from the active source.

FAQ

Common questions

How can I record a Chrome tab with audio?

If you only need the sound, open the Chrome tab that plays the audio, start Dolphin from the toolbar, keep the tab audible, stop when finished, trim the recording, and download the MP3. Dolphin does not record screen video.

Can Dolphin record both the Chrome screen and tab audio?

No. Dolphin records audio from the active Chrome tab only. Use a screen recorder if the final file must include video or on-screen actions.

Does Dolphin record all browser tabs at once?

No. Dolphin focuses on the active Chrome tab after you start recording.

Is this the same as system audio recording?

No. Dolphin records audio from the active browser tab. It does not capture full system audio or microphone input.

Can I record protected browser audio?

Some protected or DRM-based audio may not be available to browser tab capture.

Where is the browser audio file saved?

Dolphin saves the finished MP3 locally through Chrome downloads.
