使用流程

打开网页，开始录制，保存 MP3

  1. 打开有声音的标签页

    播放课程、播客、访谈、网页播放器或其他你有权保存的音频。

    录完后通过音波选择要保留的片段，再下载为 MP3。

快捷键

用快捷键更快开始和停止

你可以在 Chrome 扩展快捷键设置里为 Dolphin 绑定快捷键。按一次开始录制，再按一次停止。

推荐设置 Chrome 扩展快捷键

在地址栏打开 chrome://extensions/shortcuts，找到 Dolphin，然后设置“开始/停止录制”的快捷键。

Mac Command + Shift + Y Windows Ctrl + Shift + Y

常见问题

你可能会关心的问题

Dolphin 会录屏吗？

不会。Dolphin 只录制当前 Chrome 标签页的声音，不生成视频文件。

录音会上传到服务器吗？

不会。录音和 MP3 编码在本地完成，文件保存到你的设备。

可以录麦克风或整个电脑声音吗？

不可以。Dolphin 专注于当前浏览器标签页音频。

可以录所有网站吗？

大多数能正常播放标签页声音的网站都可以尝试录制，但受保护或 DRM 内容可能无法录制。
